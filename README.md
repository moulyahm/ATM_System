#  ATM Machine Simulation (C++)

## Description
This is a simple ATM Machine simulation program written in C++.
It allows users to:

* Deposit money
* Withdraw money (with PIN verification)
* Check balance
* View mini statement

## Features

* PIN authentication for withdrawal
* Balance management
* User-friendly menu
* Basic ATM operations

## Technologies Used

* C++
* Standard Input/Output

##  How to Run

1. Compile the program:
g++ atm.cpp -o atm

2. Run:
./atm

## Default PIN

1234

##  Sample Output

2. Withdraw
